## Development

Please set up packages first.

```console
> npm install
> npm start -s
```

### Deploy

Before deploying pages, you need to install [gh-pages](https://github.com/tschaub/gh-pages).

```console
> npm install -g gh-pages
```

To deploy,

```console
> npm run deploy
```
